@media

At-правило @media в CSS связывает набор операторов, ограниченных фигурными скобками, в CSS блок, применяется при соблюдении условия одного или нескольких медиавыражений.

Типы

all - Подходит для всех устройств.

print - Intended for paged material and for documents viewed on screen in print preview mode. Please consult the section on paged media, and the media section of the Getting Started tutorial for information about formatting issues that are specific to paged media.

screen - Предназначен в первую очередь для цветных компьютерных экранов.

speech - Предназначен для синтезаторов речи.
